# 📌 study-sample-img-react

이 프로젝트는 2022년부터 현재까지 CSS, JavaScript, GSAP 등을 활용해 제작한 예제들을 정리한 개인 아카이브입니다.
각 항목을 클릭 시 프로젝트명이 클립보드에 복사되며, 해당 이름은 GitHub 저장소와 로컬 폴더명과 동일 하게 관리하고 있습니다.
수록된 예제들은 여러 레퍼런스를 단순히 복사한 것이 아닌 분석하고 이해한 내용을 바탕으로 재구성한 결과물입니다.
또한 시간이 지난 후에도 관련 내용을 파악할 수 있도록 모든 프로젝트에 주석을 작성해 학습 기록을 하고 있습니다.
